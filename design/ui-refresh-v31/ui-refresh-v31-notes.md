# Echo UI Refresh V3.1 Notes

## 定位

V3.1 是 V3 的毛玻璃增强预览，不是新设计系统。它继续遵守 V3 的硬约束：

- 不改功能入口。
- 不改操作路径。
- 不改信息架构。
- 不新增主导航。
- 不改变按钮、菜单、弹窗的语义。

## 本次修正

上一版半透明和高斯模糊在静态预览图里不够明显。本次只增强视觉可感知度：

- 在页面底层加入连续的细纹和柔和色带，让玻璃层背后有内容可被模糊。
- 顶栏、统计页顶栏、历史页顶栏改为更明显的半透明白，并提高 `backdrop-filter` blur 强度。
- 对话页底部输入栏改为更清晰的毛玻璃层，保留附件、图片、OCR、发送入口。
- 搜索栏、对话卡片、统计卡片、设置菜单卡片增加玻璃高光边缘和轻阴影。
- 主页仍保持统一底色，不恢复顶部和列表区域的色块割裂。
- 对话页助手回复仍是开放式正文，不恢复气泡框。
- 设置菜单图标仍保持现有 Material 图标语义。

## 设计边界

- 毛玻璃只作为层级增强，不改变 Echo 原版入口和操作逻辑。
- 不做大面积换壳，不引入新的导航体系。
- 不把视觉效果应用到正式 Compose 代码；当前目录只保存预览资产。
