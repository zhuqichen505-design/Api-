# Echo UI Refresh V3.1 Notes

## 定位

V3.1 是 V3 的轻毛玻璃增强预览，不是新设计系统。它继续遵守 V3 的硬约束：

- 不改功能入口。
- 不改操作路径。
- 不改信息架构。
- 不新增主导航。
- 不改变按钮、菜单、弹窗的语义。

## 本版新增视觉效果

- 顶栏：半透明白底，轻微高斯模糊和细边线。
- 对话页输入栏：半透明底层，保留附件、图片、OCR、发送入口。
- 搜索栏：轻半透明材质，保持原位置和形态。
- 卡片：轻透明白底和弱模糊，只增加层次，不改变卡片内容。
- 统计页和历史页顶部：轻毛玻璃效果，入口仍为原结构。

## 设计边界

- 毛玻璃效果只用于界面层级，不用作装饰。
- 不使用大面积复杂背景。
- 主页背景仍保持一致。
- 对话页助手回复仍是开放式正文，不恢复气泡框。
- 设置菜单图标仍保持现有 Material 图标语义。
