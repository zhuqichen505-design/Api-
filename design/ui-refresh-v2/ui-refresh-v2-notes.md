# Echo UI Refresh V2 Notes

## 为什么重做

第一版预览完成了“简约”，但视觉完成度不够：

- 白卡片过多，界面显得像线框稿。
- 信息层级清楚，但缺少精致感和产品质感。
- 色彩使用太保守，状态、品牌和重点信息没有形成记忆点。
- 单屏之间风格一致，但缺少让用户愿意停留的视觉节奏。

## V2 方向

第二版保留简约，同时增强美观：

- 顶栏使用半透明白底和细分割线，模拟真实移动应用的轻浮层。
- 卡片圆角控制在 8px 左右，用边线、间距和轻阴影建立层级。
- 主色仍为蓝色，但加入少量青绿色过渡，用于主操作和数据状态。
- 玫红只用于缓存命中、提醒等语义状态，避免变成大面积装饰。
- 统计页加入更精致的数据卡和柱状图，让页面不再只是文字列表。
- 对话页重点优化阅读区、气泡、输入栏和上下文状态入口。
- 设置页从“卡片堆叠”改为“分组列表”，更像成熟工具应用。

## 后续落地建议

1. 先从主题层统一颜色、圆角、边线、阴影和间距 token。
2. 将主页卡片、设置项、统计卡片、消息气泡抽成可复用组件。
3. 先落地主页和对话页，因为这两个界面最影响整体观感。
4. 每个页面落地后都用截图验证文字、间距和按钮触控区域。
